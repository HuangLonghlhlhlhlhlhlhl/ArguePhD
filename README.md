# 怼人助手 (Roast Assistant) - PRO 🚀

**“机智回应，绝不破防。”**

这是一个基于 React Native + Node.js 的智能聊天助手，旨在帮助用户在职场、社交、闲聊中通过 AI 力量获取最得体（或最有杀伤力）的回应方案。

## ✨ 2026-03-21 重大版本更新 (V1.2 Premium)

### 1. 颜值革命：极简冷调 UI
*   **配色方案**：采用莫兰迪深灰色 (#F8F9FA) 与碳黑色 (#1A1A1A) 的对比，低调、犀利、有腔调。
*   **布局重构**：全屏大卡片流，悬浮操作区，24px 极度舒适圆角。
*   **图标系统**：统一了线性风格图标，界面重心更加聚焦。

### 2. 深度 AI 交互引擎 (对线实验室)
*   **双重模式**：
    *   🔥 **逻辑对线**：精准捕捉对方漏洞，降维打击。
    *   🛡️ **心态维护**：AI 心理按摩，防止被对方言论“破防”。
*   **拟人化人格**：
    *   甄嬛模式 (高冷大清风)、极客模式 (逻辑降智)、打工人模式 (反卷斗士)、毒舌模式 (火力全开)。
*   **多阶强度**：优雅 (30%)、犀利 (60%)、爆破 (100%)。

### 3. 硬核视觉功能：截图一键反击 (OCR) 🪄
*   **截图识别**：通过魔法棒图标调取相册，自动识别聊天截图内容。
*   **智能场景判断**：自动分析对方是否在进行职场 PUA 或道德绑架，并给出建议。

## 🛠️ 技术栈
*   **前端**：React Native (Expo), React Native Paper, Expo Image Picker.
*   **后端**：Node.js Express, CORS, Morgan (日志系统).
*   **API**：支持 Mock 模拟数据与真实 OCR 视觉接口对接。

## 🚀 快速启动

### 1. 启动前端
```bash
npm start
# 建议按 'w' 在浏览器预览，或按 'i/a' 在模拟器运行
```

### 2. 启动后端 (Server)
```bash
cd server
npm install
node index.js
```
